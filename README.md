# MOMentum 🏃‍♀️‍➡️

## 📖 Overview
MOMentum is an AI-assisted family management system designed to reduce the mental load of families by organising tasks, schedules, grocery lists, and communication in one unified platform.

It includes:
- Task Manager (Personal + Family)
- Family Calendar
- Grocery Tracker
- Family Messenger
- AI Assistant (AMM-AI using Ollama)

---

## 🚀 Features

### 🏠 Dashboard
- Shows today’s tasks and schedule
- Role-based view (Parent / Child)

### 📅 Family Calendar
- Daily, Weekly, Monthly views
- Filters by family member

### 📌 Task Manager
- Personal tasks (user-specific)
- Family tasks (shared)
- Mark as done / delete tasks

### 🛒 Grocery System
- Add grocery items
- Assign to family members
- Track purchase status

### 💬 Family Messenger
- Real-time message board
- Edit/delete own messages only

### 🤖 AMM-AI
- Local AI assistant using Ollama
- Answers questions based on:
  - Family tasks
  - Personal tasks
- No cloud API required

---

## 🧠 Tech Stack
- Python
- Streamlit
- SQLite
- Ollama (LLM: llama3.2 / llama3.1)
- Requests (for AI API calls)
